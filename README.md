# geturl
初步设想，把粘贴板中的图片，或者截图发到qiniu中，并获取url，发送到粘贴板中


start;
switch
    case 1:
	    ctrl alt c:
	        if img:
	        		cover to jpg or png
	            up to qiqiu
	             get the url,to paste, 
	        then:
	            ctrl v 
    case2:
	    ctrl alt x，截图
	        cover to jpg or png
	        up to qiqiu
	        get the url,to paste, 
	        then:
	            ctrl v
